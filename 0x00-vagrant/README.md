## Prerequisites
#**Install Vagrant on Mac OSx**

```
$ vagrant init ubuntu/trusty64
```
Fill in with desired base:
```
base = "ubuntu/trusty64"
```
Start your virtual machine:
```
$ vagrant up
```
Enter virtual machine:
```
$ vagrant ssh
```

# Install Git
```
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install git
```
